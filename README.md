## 使い方

![](https://cdn-ak.f.st-hatena.com/images/fotolife/b/baba_s/20190327/20190327124444.png)

「iPhoneXSafeAreaDrawerSettings」を選択して  

- IsEnable をオンにすると有効になります  
- IsPortrait をオンにすると縦持ち表示になります
